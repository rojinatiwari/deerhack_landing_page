Powerful Features

Everything you need for seamless cross-device control

Clipboard Sync

Instantly sync clipboard content between your mobile and laptop. Copy on phone, paste on desktop.

File Transfer

Transfer files between devices with chunked WebSocket transfer. No size limits, no cloud storage needed.

Remote Input

Control your laptop with touchpad-style mouse and virtual keyboard from your mobile device.

Presentation Control

Navigate slides in Google Slides, Canva, and other presentation tools with simple gestures.

Media Control

Control media playback, volume, and system audio from your mobile device.

Battery Sharing

Monitor battery levels across all connected devices in real-time.

System Components

Four powerful components working together seamlessly

Mobile App

React Native

Acts as a remote controller and data sender. Communicates with desktop/browser over WebSocket.

  • Clipboard sync
  • File transfer
  • Touchpad-style mouse
  • Virtual keyboard
  • Background support
  • QR code pairing

Desktop App

Python

Background service that listens over LAN. WebSocket server built with asyncio and websockets.

  • Clipboard receive & update
  • Mouse & keyboard simulation
  • File receiver/sender
  • Command execution
  • QR code generator
  • System tray integration

Web Controller

Browser-based

Browser-based UI for testing features without mobile app. Hosted on local network.

  • Clipboard sync (manual)
  • Slide control
  • Touchpad simulation
  • File upload
  • QR code pairing UI
  • LAN-only operation

Browser Extension

Chrome/Firefox

Injects scripts into Google Slides, Canva, Prezi. Simulates keypresses for slide navigation.

  • Site-specific scripts
  • WebSocket client
  • Slide navigation
  • Scoped permissions
  • Key event simulation
  • Future overlay UI

Security & Privacy

Secure Device Pairing

QR code-based pairing with one-time tokens ensures only authorized devices can connect.

LAN-Only Communication

All communication happens over your local network. No internet connection required.

No Third-Party Servers

Your data never leaves your network. No cloud storage or external services involved.

Command Whitelisting

Strict whitelisting of allowed commands prevents unauthorized system access.
